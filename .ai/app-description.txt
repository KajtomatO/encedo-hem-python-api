A library that exposes a simple intereface that wraps around REST api calls to HEM device.
Library - REST client
HEM device - REST serwer

Library should:
 - expose all REST endpoints.
 - provide quality-of-life features like checking state of HEM device, automaticly providing "check-in", automatic autentication.
